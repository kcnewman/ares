# ARES Documentation

This folder contains detailed project documentation beyond the root `README.md`.

## Document Map

- [`DEVELOPMENT.md`](DEVELOPMENT.md): local setup, stage-by-stage execution, and test workflow.
- [`CONFIGURATION.md`](CONFIGURATION.md): config files, environment variables, and contract notes.
- [`API.md`](API.md): FastAPI endpoint reference, request/response examples, and local usage.
- [`OPERATIONS.md`](OPERATIONS.md): runbook for training, packaging, Docker, and Cloud Run deployment.
- [`TROUBLESHOOTING.md`](TROUBLESHOOTING.md): common failures and diagnostic steps.

## Audience Guide

- New contributor: start with `DEVELOPMENT.md`.
- Application integrator: start with `API.md`.
- Operator/maintainer: start with `OPERATIONS.md`.
- Debugging issues: use `TROUBLESHOOTING.md`.
